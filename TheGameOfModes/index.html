The Game of Modes project

Text

"Barcelona, perhaps rising from their ruins again"

Implementation

This mode is implemented with the <blockquote> element, which semantically highlights quotations or key statements. An alternative would be to use regular <p> tag for default text.

Effect on the reader

Using a blockquote draws the eye of the reader and signals significant or important information, giving the text more weight than a normal text paragraph. It can create a pause in the normal reading flow which can make the reader reflect more deeply on the words being emphasized.

Photo

Implementation

This mode is implemented with a <figure> element containing an <img> and a <figcaption>. This semantically links the image with its caption which in this case is used to provide the source information. Alternatives include just using a simple <img> tag or presenting multiple images in a gallery or slideshow component.

Effect on the reader

Images are processed instantly and evoke emotions faster than text. They boost attention, increase credibility, and make information more memorable.

Audio

Music by Oleksandr Savochka from Pixabay

Implementation

Implemented with the <audio> tag. Alternatives: embedded players like Spotify, YouTube, or Apple Music.

Effect on the reader

Sound enhances how the reader perceives and experiences the content. Background audio can set the right atmosphere and help create a good environment.

Video

Video by Kevin Krämer from Pixabay

Implementation

Implemented with the HTML <video> tag using a local MP4 file. Alternatives: embed from YouTube/Vimeo with an <iframe> or custom video players like Video.js or Plyr.

Effect on the reader

Video engages the reader through movement, making the story feel more alive and makes it easier to remember. Motion draws the attention of the reader and keeps them interested in a way static images can´t.

Interactive map

Implementation

This mode is implemented with a simple <iframe> embed from Google Maps. Alternatives: Google Maps JavaScript API for more customization, or libraries like Leaflet or Mapbox for more control.

Effect on the reader

An interactive map captures attention by combining visual information with user interaction. It invites exploration and helps readers connect information with a real location, making it more engaging and memorable.

Implementation

Structured as a semantic list styled with CSS to form a visual timeline. Alternatives: use TimelineJS or Vis.js for interactive timeline

Effect on the reader

Timelines easily simplify information and present it in a clear and easy-to-read context. Visual anchors (dots/lines) help readers remember specific times or key dates important to the story.

Infoviz

Implementation

This visualization is a QuickChart doughnut rendered via an <img> src wrapped in a semantic <figure>/<figcaption>. The palette matches the site theme, and the alt text summarizes the content for good accessibility. Alternatives: an interactive Chart.js/D3 version (tooltips/filters) or an edited static image.

Effect on the reader

The infovis turns numbers and statistics into easy-to-read shapes and colors. It makes the information clear at a glance and anchors the message with a clear caption.
